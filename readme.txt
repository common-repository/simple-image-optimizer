Free plugin for image optimization. Simple Image Optimizer lets you set the quality of all images to to improve your SEO.

== Description ==
Free plugin for image optimization.

Set the quality of all images with one click, you can go back if you think the quality set is too much. Also, the original images are not erased.

Improve your SEO! Make your site faster. 

Please comment if you find the plugin useful or if you have any inquiries.

== Installation ==
1. Upload the “simple-image-optimizer” plugin to your /wp-content/plugins/ directory.
2. Activate the plugin.
3. Go to settings -> \"Simple Image Optimizer\" and set the desired quality for all images. 
4. Take a look to your NEW site speed!!
